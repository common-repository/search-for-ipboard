Allows you to search for IPS Community Suite (former IP.Board) topics and posts right from WordPress.

== Description ==

If you have an IPS Community Suite (former IP.Board) based forum, this extension is a must. It'll allow your vititors to search for forum posts and topics right from your Wordpress site. A lot of options, available in Premium version, will also give you full control on search process.

**STANDARD FEATURES**

- Compatible with all versions of IPS Community Suite 4.
- Allows you to search for IP.Board (IPS Community) content from WordPress.
- Follows IP.Board SEF URLs structure.
- Applies all necessary Wordpress filters to search results.
- Template independent.

**PREMIUM FEATURES**

- Limit number of search results (useful for large databases).
- Sort results by topic title, number of views or post date.
- Search for exact phrase or any keyword from the list.
- Position results before or after Wordpress posts. Or even replace them!
- Limit search by topic titles only (useful for large databases).
- Search in member names as well.
- Include closed topics in search results.
- Show or hide parent Forum category before topic name.
- Set custom thumbnail for search results.
- Exclude forums from search results or include only certain forums.

You can purchase Premium version <a href="https://thekrotek.com/wordpress-extensions/ipboard-search">here</a>.

== Installation ==

1. Unpack and upload all files to "/wp-content/plugins/plugin-name" directory. Or simply install plugin through Plugins page.
2. Activate extension on "Plugins" page.
3. Setup plugin according to your liking.
